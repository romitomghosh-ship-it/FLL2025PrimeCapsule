# FLL 2025 Prime Capsule
This repository has code and instructions for building a temperature control system that can be added to a box for holding artifacts.

This code uses the following hardware:
1. Raspberry Pi Pico 2W
2. DHT11 temperature/humidity sensor
3. LCD I2C Display
4. L298N Motor Driver
5. 5V Fan
6. Battery pack
